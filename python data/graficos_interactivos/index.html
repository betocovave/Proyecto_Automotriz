Visualizaciones Interactivas - Talleres Automotrices

En esta página puedes acceder a todas las visualizaciones interactivas generadas para el análisis de talleres automotrices. Cada visualización te permite explorar los datos de diferentes maneras, incluyendo filtros, hover para más información y zoom.

Visualizaciones Disponibles

  • Eficiencia vs Reparaciones por Tipo de Taller

    Visualiza la relación entre la eficiencia y el número de reparaciones realizadas. El tamaño de los puntos representa el número de mecánicos disponibles.

  • Capacitación vs Tiempo de Reparación vs Productividad

    Explora cómo el nivel de capacitación de los técnicos influye en el tiempo de reparación. El tamaño de los puntos representa el número de reparaciones realizadas.

  • Visualización 3D de Eficiencia, Capacitación y Reparaciones

    Gráfico tridimensional que muestra la relación entre eficiencia, capacitación de técnicos y reparaciones realizadas. Puedes rotar, hacer zoom y filtrar por tipo de taller.

  • Dashboard Completo de Talleres

    Panel interactivo que incluye una tabla de datos y múltiples gráficos comparativos entre los diferentes tipos de talleres.

  • Tabla Completa de Datos

    Visualiza todos los datos del dataset en formato de tabla interactiva. Puedes ordenar, filtrar y explorar todos los 65 registros con sus 11 variables.

Nota: Estas visualizaciones son interactivas. Puedes hacer clic, arrastrar, hacer zoom, filtrar y pasar el cursor sobre los elementos para ver información adicional.
